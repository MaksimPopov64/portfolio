# Портфолио Frontend-разработчика

Статический сайт-портфолио для GitHub Pages.

## Особенности
- Современный адаптивный дизайн
- Карточки проектов с детальной информацией
- Секция для скачивания резюме в PDF
- Оптимизировано для мобильных устройств
- Чистый и минималистичный интерфейс

## Технологии
- HTML5
- CSS3 (Flexbox, Grid, CSS Variables)
- Vanilla JavaScript
- Font Awesome иконы
- Google Fonts (Inter)
